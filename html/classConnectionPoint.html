Detailed Description

A small circle on edge of a node used for connection.

The connection point can be either input or output: it can be either on the right or on the left.

Depending on this, it responds differently to user actions (hovering, mouse clicks) and indicates whether it is ready for interaction. The exact behavior is defined in the child classes
