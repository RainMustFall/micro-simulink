Detailed Description

Buffer which is used to pass a visual node from Palette to GraphWidget, when user drags it with mouse.

Usually QMimeData is used to store the data that is being dragged. The problem is that it supports only text data and is not very useful for the hierarchy of nodes we have here.

Instead of QMimeData, we have a small helper class that temporarily owns the node that is being dragged. If drag succeeds, the ownership is transferred to the GraphWidget. If not, it is discarded and destroyed as soon as another drag starts.
