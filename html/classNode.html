Detailed Description

template<typename T>
class Node< T >

Represents a node of the graph (input, output or intermediate).

It allows to attach/detach dependencies (other nodes) and return the execution result (calling Execute from its dependencies, if necessary).

It is a template parameterized with a type T - calculations are performed on this type. Inheriting classes override the Execute method with specific actions on an object of type T. An example of a T type could be a function of one variable, a numeric constant or a LaTeX expression. In general, it is a type that supports specific arithmetic operations (such as the addition operator).
